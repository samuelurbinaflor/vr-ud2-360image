# vr-ud2-360image
Reto: Llevando la realidad a una experiencia VR

En este reto vas a montar una experiencia VR con imágenes y vídeos en 360 grados.

Se van a evaluar los siguientes criterios:

2c) Se han elaborado escenarios y experiencias en 360 grados.

2d) Se ha realizado la grabación de contenidos en 360 grados.

Estos son los pasos a dar:

1. Captura rincones del instituto (imágenes o vídeos) con una cámara 360º.

2. De forma opcional, genera otras imágenes usando el Google Colab de 360PanT

3. De forma opcional, genera otras imágenes usando WorldLabs.

4. Crea una experiencia VR usando A-Frame y pruébala en VR. Ejemplo. Os adjunto también una página en la que se pueden pasar las escenas hacia delante y atrás usando los botones de los controllers de las Meta Quest.

Entrega:

- Un enlace a la Git Page de la experiencia.

- Un enlace a un vídeo breve probándola en VR.
